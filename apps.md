# GPU Apps — Product Vision

## Proactive Assistant (Вариант C)

Клавиатура/системный слой который замечает паттерны в тексте и предлагает действия.

### Принципы

- Модель видит ограниченно (с согласия юзера)
- Только высокоуверенные паттерны (>85%, первые 2 недели >95%)
- Тонкая ненавязчивая полоска после отправки сообщения (не popup, не modal)
- Адаптируется под юзера (учится когда молчать)
- Логические цепочки, не просто regex ("за 2 часа до рейса" → вычисляет время)
- Fallback на полную апку для сложного

### Адаптация порога

- Юзер отклонил подсказку → порог для этого паттерна +5%
- 3 отклонения подряд → этот тип подсказок выключается
- Юзер принял → порог -2%

### Privacy

- Всё на устройстве, контекст никогда не уходит на сервер
- Поля secureTextEntry (пароли) → модель не видит
- Юзер может пометить любую апку/чат как "приватная"
- Всегда показываем ЧТО модель увидела, всегда даём кнопку "не читай этот чат"

### Архитектура (iOS)

- Клавиатура (lightweight, 30 MB): автокомплит + паттерн-матчинг (regex)
- Главная апка (heavyweight): полная модель, чат с экспертами, скилл-стор
- Связь через App Group

### Отличие от Apple Data Detectors

Не конкурируем на уровне "подчеркнуть дату". Конкурируем на уровне логических цепочек:
- "рейс в 18:30, надо быть за 2 часа" → такси к 16:00
- "давай в пятницу, но не раньше обеда" → событие пт 13:00

## GPU Chat — ИИ + живые эксперты

### Поток разговора

Один поток, не два экрана. ИИ отвечает сразу и бесплатно. Если не уверен — предлагает эксперта. Эксперт приходит в тот же разговор.

### Уровни уверенности

- 80%+: только ответ ИИ, без предложения эксперта
- 50-80%: ответ ИИ + маленькая ссылка "спросить эксперта"
- <50%: ответ ИИ + явное предложение с карточками экспертов
- <20%: "Я не знаю. Вот кто может помочь:"

### Модель учится на виду

После разговора с экспертом GPU показывает: "Запомнил: [вывод]". Юзер видит рост модели и может поправить.

### Fallback-цепочка

1. ИИ знает → отвечает бесплатно
2. Есть скилл в каталоге → вызывает скилл
3. Есть эксперт онлайн → предлагает чат (платно)
4. Нет никого → заказ в гильдию умельцев

### Монетизация

- ИИ-ответы бесплатно
- Консультация с экспертом — платная ($1-10/ответ)
- Комиссия платформы 15-20%
- Эксперты зарабатывают, модель учится, юзер получает ответ

## GPU Skill Store — маркетплейс AI-скиллов

### Кто создаёт скиллы

- Вайбкодеры/билдеры через Skill Builder (без ML-знаний)
- Описание задачи + 20 примеров → автоматический LoRA → публикация

### Гильдия умельцев

- Юзеры размещают заказы на недостающие скиллы
- Умельцы берут заказы, создают скиллы, получают награду
- Скилл становится доступен всем

## Enablers для билдеров

### GPU SDK (5 базовых функций)

```
gpu.complete(prompt)           — генерация текста
gpu.translate(text, from, to)  — перевод
gpu.extract(text, schema)      — structured extraction
gpu.summarize(text, maxLength) — суммаризация
gpu.embeddings(text)           — вектор для поиска
```

Без ключей, без регистрации, без оплаты. Модель на устройстве юзера.

### Intent API

Билдер описывает схему интентов → модель парсит юзерский ввод → temperature=0 + constrained decoding = детерминированный результат.

### Платформенные принципы

- AI как public utility (как GPS — просто есть на устройстве)
- "Принеси свой эксперт" (юзер = compute node в сети)
- Федерация, не корпорация (нельзя отключить, нет vendor lock-in)
- Антихрупкость (больше нагрузки → больше нод → мощнее сеть)
- Zero-cost AI для бедных стран
- AI без интернета (Bluetooth mesh, peer-to-peer Wi-Fi)
