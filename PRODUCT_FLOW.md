# Jippy — Полный продуктовый флоу

## Миссия (показываем пользователю)

Мы, человечество, учим свою распределённую ИИ. В каждом телефоне живёт маленький ИИ, который разбирается в какой-то теме лучше всего. Вместе эти маленькие ИИ образуют коллективный разум — мощнее любого суперкомпьютера. Ты — часть этого.

### Ежедневное использование

**Клавиатура:**
- Подсказывает слова и предложения
- При каждом новом слове по теме — хлопушка: "🎉 +1 XP к Кулинарии!"
- Фоново учится на всём что пишешь

**Кормление (главная механика роста):**
- Выделил текст на любом сайте → кнопка "Скормить Jippy" (через iOS Share Sheet)
- Скопировал ссылку → Jippy заходит по ней, скрапит всю страницу, съедает текст
- Загрузил PDF / файл → Jippy переваривает
- После каждого кормления:
  - Jippy оценивает новизну данных
  - "Новая информация! +25 XP 🎉" или "Уже знаю это, +2 XP"
  - 🔔 "Jippy переварил данные! Можно кормить снова!"

**Вопрос-ответ (тоже обучение!):**
- Спрашиваешь Jippy → он отвечает
- Если ответ неправильный → поправляешь → Jippy учится: "+3 XP"
- Если ответ правильный → подтверждаешь 👍 → закрепление знания

**Если Jippy не знает ответ:**
- Тихо спрашивает сеть (другие телефоны или серверную модель)
- Пользователь видит просто ответ
- Jippy запоминает ответ → в следующий раз ответит сам

### Система уровней

```
Бесконечная прокачка (как Diablo skill tree):

Уровень 1 "Малыш"       — Базовые знания по теме. Не зарабатывает.
Уровень 2 "Ученик"       — Знает основы. Не зарабатывает.
Уровень 3 "Стажёр"       — Отвечает на простые вопросы. 1× JC.
Уровень 4 "Специалист"   — Обошёл серверную модель по теме! 3× JC. 🎉
Уровень 5 "Эксперт"      — Топ-10% в сети. 10× JC.
Уровень 6 "Мастер"       — Топ-1% в сети. 30× JC.
Уровень 7+ "Легенда"     — Бесконечная прокачка. 50×+ JC.
...нет потолка!

Ключевой принцип: прокачка НИКОГДА не заканчивается.
Можно одновременно:
  • Углублять текущую ветку (Грузинская кухня → Хинкали →
    Пхалийский хинкали vs Калакурский хинкали)
  • Открывать новые ветки (Кулинария + История + Музыка)
  • Качать предыдущие уровни глубже (даже Уровень 3
    можно прокачивать до бесконечности)

Как в Diablo: ты можешь вкладывать очки и в старые
скиллы, и в новые ветки. Распределяешь сам.
Нет максимального уровня — всегда есть куда расти.
```

Механика Boss Battle:
- 10 раундов, каждый = 1 вопрос по теме пользователя
- Оба отвечают вслепую, нейтральный судья оценивает
- Победа = XP бонус + трофей + разблокировка следующего босса
- Боссы по сложности: Gemma → GPT-4o → Claude → топ Jippy в сети
- Можно перебивать (rechallenge) после прокачки
- Проигрыш = "Нужно ещё покормить! +5 XP за смелость"

**Оценка уровня — двухступенчатая система (из эксперимента FEEDING-001/002):**

1. **Примерный уровень** (по объёму данных):
   Показываем прогресс-бар на основе количества скормленных данных.
   Из эксперимента: ~100 примеров ≈ приближение к серверу, ~500+ ≈ уверенное превосходство.
   Пользователь видит: "До следующего уровня: ещё ~87 XP (примерно 40 текстов)"

2. **Подтверждённый уровень** (слепой бенчмарк):
   Перед повышением на Уровень 4+ сервер запускает автоматический бенчмарк:
   20+ вопросов по теме, телефон и сервер отвечают вслепую, сравниваем.
   Порог: модель побеждает в >60% вопросов → уровень подтверждён.

**Важно:** данные для обучения должны быть ДВУНАПРАВЛЕННЫМИ (из FEEDING-003):
- "Ткемали — это соус из алычи" (прямое)
- "Кислый соус из алычи — это ткемали" (обратное)
- "Вопрос: Из чего делают ткемали? Ответ: Из алычи" (Q&A)
Без этого модель описывает, но не определяет.

```

    │

    │   │

    │

    │
    ├── Найден телефон-эксперт (Маша, Мастер кулинарии)

    │

```
