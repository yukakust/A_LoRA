### Recommended Scraping Priority (for maximum text data):
1. **RussianFood.com** (1,511 recipes) - highest single-site count
2. **Povar.ru** (612 recipes) + **Eda.ru** (622 recipes) - rich step-by-step text
3. **RecipeNLG/Recipe1M+** datasets - filter for Georgian/Caucasian
4. **Wikipedia** all languages - clean structured text
5. **Free e-books** from royallib/rulit/aldebaran - complete books
6. **GeorgianRecipes.net/.com** - English-language dedicated sites
7. **Kulinaria.ge** + **Gurmania.ge** - native Georgian text
8. **Cultural articles** + **blogs** - supplementary data
